# Cloudant

Ruby Cloudant is a Ruby wrapper for [IBM Cloudant's](https://cloudant.com/) API. Cloudant is a NoSQL database built on [CouchDB](http://couchdb.apache.org/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cloudant'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cloudant
